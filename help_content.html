Pet project

для создания и проведения любительских турниров видов спорта, в которых победитель матча определяется по итогам 3-х сетов.

Роли пользователей

Пользователь может быть:

  • администратором турниров
  • участником
  • зрителем
  • доверенным участником (в следующих версиях будет добавлена роль)
  • счетоводом (в следующих версиях будет добавлена роль)
  • судьей (в следующих версиях будет добавлена роль)
Администратор турниров

Может:

  • создавать турниры
  • добавлять в них игроков
  • вносить результаты матчей
Как стать администратором турниров:
  1. Нужно получить пароль по расположенной на главной странице кнопке "Получить пароль". В открывшейся форме нужно вести имя, email. Реальность email не проверяется, но если email валидный "админ всей системы" сможет по запросу отправить на него пароль.
  2. После получения пароля доступен вход по кнопке "Админ турнира". В открывшейся форме нужно ввести email и пароль.
Управление турнирами:
  • Страница "Управление турнирами описание кнопок".
  • Кнопка "Создать турнир" открывает форму настроек турнира с предустановками для Бадминтона, Волейбола, Пинг-понга. Текущая логика настроена для матчей из трех сетов до двух побед.
  • После создания турнира он появляется в списке турниров для данного Админа турниров.
  • По кнопке "Просмотр" открывается страница турнира с его названием, видом спорта и т.д. После создания редактировать настройки турнира пока нельзя.
В верхней части страницы три кнопки:
  • Кнопка "Добавить участника" позволяет добавить участника:
    • Вручную.
    • Из списка (в списке имена тех кто когда-то был добавлен).
    • Из листа ожидания (в листе те кто выбрал турнир из списка и подал заявку самостоятельно).
Важные особенности:
  • После введения результата любого матча турнира кнопка "Добавить участника" исчезает. Нужно ли предусмотреть возможность добавления опоздавших покажет тестирование.
  • Кнопка "Составить расписание заново" удаляет все результаты и пересоставляет расписание. После добавления всех участников нужно проверить правильно ли составлено расписание так как иногда автоматика глючит.
  • Кнопка "Выгрузить в Excel" - сохраняет данные турнира в виде файла "tournament_название_турнира_yyyymmdd_hhmm.csv".
  • Для ввода результата кликнуть по клеточке в "Турнирной таблице" или на иконку редактировать в расписании.
  • Введите нужный счет. По кнопке "Сохранить" счет будет сохранен в турнирной таблице и в расписании. Автоматически будут начислены очки, определены места участников и таблица "Статистика участников" будет отсортирована с учетом набранных очков, личных встреч и разницы очков в сетах.
Завершение турнира:
  • После заполнения данных для всех матчей в турнире автоматически будет определен и поздравлен победитель.
Определение победителей:

Места участников в турнире определяются по следующим критериям (в порядке приоритета):

  1. Количество набранных очков - участник с большим количеством очков занимает более высокое место.
  2. Результат личной встречи - при равенстве очков у двух участников, выше место у того, кто победил в личной встрече.
  3. Разница выигранных и проигранных сетов - при равенстве очков у трех и более участников, выше место у того, у кого лучше разница между выигранными и проигранными сетами.
  4. Разница очков в сетах - при равенстве всех предыдущих критериев, выше место у того, у кого лучше суммарная разница очков во всех сетах.
Кнопка "Ваши турниры"

Предназначена для участников турниров. Кнопка позволяет посмотреть список турниров в которых принимает игрок.

  • Для входа достаточно ввести имя игрока.
  • Открывается список турниров в которых принимает данный игрок.
  • По кнопке "Просмотр" открывается турнир в режиме просмотра без возможности редактировать.
Кнопка "Все турниры"

Открывает список турниров созданных всеми Админами турниров.

  • Для всех турниров доступна кнопка "Я зритель".
Дополнительная информация
  • Если турнир не начался для турнира доступна кнопка "Подать заявку".
  • Заявка будет видна администратору турнира.
