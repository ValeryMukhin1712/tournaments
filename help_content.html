Турнирный Администратор

для любительских турниров в которых победитель матча определяется по итогам 3-х сетов до 2-х побед.

Роли пользователей
  • Администратор турниров
  • Участник
  • Зритель
Администратор турниров
  • создаёт турниры
  • вносит результаты матчей
Как стать администратором турниров?
  1. Получите пароль по кнопке "Получить пароль". В открывшейся форме ведите имя, email. Реальность email не проверяется, но если email валидный на него придёт письмо с паролем.
  2. С паролем войдите в "Управление турнирами" по кнопке "Админ турнира".
Управление турнирами:
  • Кнопка "Создать турнир" открывает форму настроек турнира. Введите имя турнира, описание если нужно и выберите вид спорта. Для Бадминтона, Волейбола, Пинг-понга есть предварительные настройки. Текущая логика настроена для матчей из трех сетов до двух побед.
  • После создания турнира он появляется в списке турниров для данного Админа турниров.
  • По кнопке "Просмотр" открывается страница турнира с его названием, видом спорта и т.д.
На странице турнира в верхней части страницы три кнопки:
  • Кнопка "Добавить участника" для добавления участников:
    • Вручную.
    • Из списка (в списке имена тех кто когда-то уже добавлен).
    • Из листа ожидания (в листе те кто подал заявку через страницу "Для зрителей").
Нюансы:
  • Редактировать название турнира и его настройки нельзя.
  • После введения результата любого матча турнир считается начавшимся и кнопка "Добавить участника" исчезает.
  • Кнопка "Составить расписание заново" удаляет все результаты и пересоставляет расписание. После добавления всех участников желательно проверить правильно ли составлено расписание.
  • Кнопка "Выгрузить в Excel" - сохраняет данные турнира в виде файла "tournament_название_турнира_yyyymmdd_hhmm.csv".
Ввод результатов матчей:
  • Кликните по клеточке в "Турнирной таблице" или на строчку матча в расписании. Откроется форма "Результат". Введите счёт матча или Откройте страницу "Судейство" для ведения счёта Онлайн. Счёт будет автоматически обновляться на странице турнира и на страницах зрителей.
Страница Судейство:
  • Показывает схему корта и кнопки для настройки расположения команд и места начальной подачи.
  • В центре корта кнопка обнулить счёт
  • В левой части функциональные кнопки для:- Выхода из режима судейства, Демонстрации счёта игрокам, Отмены последнего розыгрыша, Перехода на команды другую сторону корта, Смены позиций игроков в левой и правой команде.
  • Клик по левой или правой половине корта добавляет очко соответствующей команде. Счёт автоматически обновляется на турнирной странице и на страницах зрителей. Результат розыгрыша фиксируется в Журнале матча.
Завершение турнира:
  • После внесения результатов всех матчей турнир завершается и автоматически определяется победитель.
Логика определения победителей:

Места участников в турнире определяются по следующим критериям (в порядке приоритета):

  1. Количество набранных очков - участник с большим количеством очков занимает более высокое место.
  2. Результат личной встречи - при равенстве очков у двух участников, выше место у того, кто победил в личной встрече.
  3. Разница выигранных и проигранных сетов - при равенстве очков у трех и более участников, выше место у того, у кого лучше разница между выигранными и проигранными сетами.
  4. Разница очков в сетах - при равенстве всех предыдущих критериев, выше место у того, у кого лучше суммарная разница очков во всех сетах.
Кнопка "Для игроков"

Предназначена для участников турниров. Кнопка показывает список турниров в которых принимает игрок.

  • Вход по имени игрока.
  • Открывается список турниров в которых принимает данный игрок.
  • По кнопке "Просмотр" открывается турнир в режиме просмотра без возможности редактировать. В расписании выделены матчи данного игрока.
Кнопка "Для зрителей"

Открывает список всех турниров.

  • Для всех турниров доступна кнопка "Я зритель". Откроется страница турнира в режиме просмотра.
  • Если турнир не начался для него доступна кнопка "Подать заявку".
  • Заявка попадёт в лист ожидания турнира.
