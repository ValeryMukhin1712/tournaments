1. Обзор приложения

Основные возможности
  • Создание и управление турнирами - полный контроль над турнирами
  • Круговая система - каждый участник играет с каждым
  • Интерактивная шахматка - удобный ввод результатов
  • Автоматическое расписание - с учетом количества площадок
  • Статистика участников - победы, поражения, очки
  • Система ролей - разные уровни доступа
  • Список игроков - база данных всех участников
  • Лист ожидания - для переполненных турниров

2. Начало работы

Шаг 1: Доступ к приложению

Откройте браузер и перейдите по адресу: http://localhost:5000

Шаг 2: Выбор роли

На главной странице выберите свою роль:

  • Участник - для просмотра турниров и подачи заявок
  • Доверенный участник - для ввода результатов матчей
  • Администратор - для полного управления турнирами
Шаг 3: Получение доступа

Для получения доступа к турниру:

  1. Нажмите "Получить доступ"
  2. Введите email и имя
  3. Получите код доступа по email
  4. Введите код для входа в систему
Важно: Для первого входа в систему используйте email admin@system с любым кодом доступа.

4. Управление турнирами

Создание турнира (только для администраторов)

Шаг 1: Переход к созданию

На главной странице нажмите "Создать турнир" или перейдите в админ-панель

Шаг 2: Заполнение данных
  • Название турнира - уникальное имя
  • Вид спорта - настольный теннис, бадминтон или волейбол
  • Дата начала - дата проведения турнира
  • Время начала - время первого матча
  • Время окончания - время последнего матча
  • Количество площадок - для составления расписания
  • Длительность матча - в минутах
  • Перерыв между матчами - в минутах
Шаг 3: Сохранение

Нажмите "Создать турнир" для сохранения

Просмотр турниров

Список турниров

На странице "Турниры" отображаются все доступные турниры с информацией:

  • Название и вид спорта
  • Дата и время проведения
  • Количество участников
  • Статус турнира
  • Кнопки для подачи заявки или входа

5. Управление участниками

Добавление участников (только для администраторов)

Способ 1: Ручной ввод
  1. Откройте турнир
  2. Нажмите "Добавить участника"
  3. Выберите вкладку "Ручной ввод"
  4. Введите имя участника
  5. Выберите тип (индивидуальный/команда)
  6. Нажмите "Добавить"
Способ 2: Из списка игроков
  1. Откройте турнир
  2. Нажмите "Добавить участника"
  3. Выберите вкладку "Добавить из списка"
  4. Найдите нужного игрока в списке
  5. Отметьте галочкой или нажмите "Добавить"
Способ 3: Из листа ожидания
  1. Откройте турнир
  2. Нажмите "Добавить участника"
  3. Выберите вкладку "Лист ожидания"
  4. Просмотрите заявки
  5. Нажмите "Принять" для добавления участника
Автоматическое расписание: При добавлении участников расписание матчей создается автоматически.

Удаление участников

Удаление из турнира
  1. Откройте турнир
  2. В таблице участников найдите нужного
  3. Нажмите кнопку "Удалить"
  4. Подтвердите удаление

6. Матчи и результаты

Просмотр матчей

Шахматка результатов

На странице турнира отображается интерактивная таблица (шахматка) со всеми матчами:

  • По горизонтали и вертикали - участники
  • На пересечении - результат матча
  • Клик по ячейке открывает форму ввода результата
  • Завершенные матчи выделены цветом

Ввод результатов (доверенные участники и администраторы)

Шаг 1: Открытие формы

Кликните на ячейку с матчем в шахматке

Шаг 2: Заполнение результата
  • Сет 1: Счет первого сета (например, 21:19)
  • Сет 2: Счет второго сета
  • Сет 3: Счет третьего сета (если нужен)
  • Статус: Завершен, В процессе, Запланирован
Шаг 3: Сохранение

Нажмите "Сохранить результат" для сохранения

Внимание: Результаты автоматически обновляют статистику участников и турнирную таблицу.

Редактирование результатов

Изменение результата
  1. Кликните на ячейку с уже введенным результатом
  2. Измените данные в форме
  3. Нажмите "Сохранить изменения"

7. Расписание

Просмотр расписания

Расписание матчей

На странице турнира отображается расписание с информацией:

  • Номер матча - порядковый номер
  • Время - время начала матча
  • Площадка - номер площадки
  • Участники - кто играет
  • Счет - текущий результат
  • Статус - запланирован, в процессе, завершен

Автоматическое составление расписания

Генерация расписания
  1. Добавьте участников в турнир
  2. Нажмите "Составить расписание заново"
  3. Система автоматически создаст расписание
Умное расписание: Система учитывает количество площадок, длительность матчей и перерывы между ними.

Фильтрация расписания

Фильтры
  • По дате - просмотр матчей на конкретную дату
  • По площадке - матчи на определенной площадке
  • По статусу - запланированные, завершенные, в процессе

8. Статистика

Турнирная таблица

Статистика участников

Автоматически обновляемая таблица с показателями:

  • Место - позиция в турнире
  • Участник - имя участника
  • Игр - количество сыгранных матчей
  • Побед - количество побед
  • Ничьих - количество ничьих
  • Поражений - количество поражений
  • Очки - набранные очки
  • Разница очков - разница в сетах

Система подсчета очков

Начисление очков
  • Победа: 2 очка
  • Ничья: 1 очко
  • Поражение: 0 очков
Определение места

Места определяются по:

  1. Количеству очков (по убыванию)
  2. Разнице очков в сетах (по убыванию)
  3. Алфавитному порядку имен (при равенстве)

Сортировка таблицы

Интерактивная сортировка

Кликните по заголовку колонки для сортировки:

  • По месту в турнире
  • По количеству игр
  • По количеству побед
  • По количеству очков
  • По разнице очков

9. Список игроков

Автоматическое ведение списка

База данных игроков

Система автоматически ведет список всех игроков, которые когда-либо участвовали в турнирах:

  • Имя игрока
  • Дата первого участия
  • Дата последнего участия
  • Автоматическое обновление при добавлении участников

Использование списка игроков

Добавление из списка
  1. Откройте турнир
  2. Нажмите "Добавить участника"
  3. Выберите вкладку "Добавить из списка"
  4. Найдите игрока в списке
  5. Отметьте галочкой или нажмите "Добавить"
Поиск игроков
  1. В поле поиска введите имя игрока
  2. Список автоматически фильтруется
  3. Выберите нужного игрока
Совет: Список игроков помогает быстро добавлять участников, которые уже участвовали в других турнирах.

10. Лист ожидания

Подача заявки на участие

Шаг 1: Выбор турнира

На странице "Турниры" найдите нужный турнир и нажмите "Подать заявку"

Шаг 2: Заполнение заявки
  • Введите ваше имя
  • Выберите тип участия (индивидуальный/команда)
  • Нажмите "Подать заявку"
Шаг 3: Ожидание рассмотрения

Администратор турнира рассмотрит вашу заявку и примет решение

Управление листом ожидания (только для администраторов)

Просмотр заявок

В турнире на вкладке "Лист ожидания" отображаются все заявки:

  • Имя заявителя
  • Тип участия
  • Дата подачи заявки
  • Статус заявки
Принятие заявки
  1. Откройте турнир
  2. Перейдите на вкладку "Лист ожидания"
  3. Нажмите "Принять" рядом с заявкой
  4. Участник автоматически добавится в турнир
Отклонение заявки
  1. Нажмите "Отклонить" рядом с заявкой
  2. Заявка будет помечена как отклоненная

11. Решение проблем

Часто встречающиеся проблемы

Проблема: Не могу войти в систему

Решение:

  • Проверьте правильность email и кода доступа
  • Убедитесь, что код не истек
  • Попробуйте запросить новый код доступа
  • Для первого входа используйте admin@system
Проблема: Не отображается расписание

Решение:

  • Убедитесь, что в турнире есть участники
  • Нажмите "Составить расписание заново"
  • Проверьте настройки турнира (время, площадки)
Проблема: Не могу добавить участника

Решение:

  • Убедитесь, что у вас есть права администратора
  • Проверьте, что участник с таким именем еще не добавлен
  • Попробуйте добавить через другую вкладку (ручной ввод/список/лист ожидания)
Проблема: Не сохраняется результат матча

Решение:

  • Убедитесь, что у вас есть права на ввод результатов
  • Проверьте правильность формата счета (например, 21:19)
  • Убедитесь, что матч не завершен другим пользователем

Техническая поддержка

Получение помощи

Если проблема не решается:

  • Очистите кэш браузера
  • Попробуйте другой браузер
  • Перезагрузите страницу
  • Проверьте подключение к интернету
